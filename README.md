# HAB-Project

### Initialization
To start backend server:
- `pip install -r  requirements.txt` (can choose to do this in an environment)
- `cd backend`
- `uvicorn app.main:app --reload`

To start frontend server: 
- `cd frontend`
- `npm install`
- `npm run dev` or `npm start`

go to `http://localhost:3000/`
